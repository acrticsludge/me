# .gitignore Template — Next.js + Worker

Copy this into `.gitignore` at the project root.

```gitignore
# See https://help.github.com/articles/ignoring-files/ for more about ignoring files.

# dependencies
/node_modules
/.pnp
.pnp.*
.yarn/*
!.yarn/patches
!.yarn/plugins
!.yarn/releases
!.yarn/versions

# testing
/coverage

# next.js
/.next/
/out/

# production
/build

# misc
.DS_Store
*.pem

# debug
npm-debug.log*
yarn-debug.log*
yarn-error.log*
.pnpm-debug.log*

# env files (NEVER commit these)
.env*

# vercel
.vercel

# typescript
*.tsbuildinfo
next-env.d.ts

# graphify knowledge graph output (generated artifact)
/graphify-out

# personal knowledge base
/me
```

## Notes
- `.env*` glob catches `.env`, `.env.local`, `.env.production`, `.env.development` — all of them
- `/graphify-out` — generated by the graphify tool; rebuild anytime with `/graphify`
- `/me` — personal knowledge base; gitignored so it doesn't clutter the repo
- The worker subdirectory has its own `.gitignore` for worker-specific `.env`
